## Ideas

### #001 - CozyFocus: Animated Study Environment

**Summary**  
A todo website with integrated calendar, timer, time tracking, and social study features wrapped in a uniquely cozy, animation-rich UI.

**Core Features**
- Todo list management
- Calendar integration
- Timer & time tracker
- Time spent analytics by subject/category
- User accounts system
- Peer competition features
- Real-time study together sessions

**Design Vision**
- Heavy on animations, blur effects, and visual uniqueness
- Cozy, warm aesthetic that adapts to user's work context
- Immersive environment matching the task at hand

**Status**: Proposed - *More features to be defined*

**Notes**  
*This idea is still evolving. Additional features will be added later.*

### #002 - EduHub: Unified Learning Platform

**Summary**  
A unified website platform connecting teachers and students for seamless assignment management, resource sharing, and academic collaboration.

**Core Features**
- Teacher portal for uploading assignments, marks, and notes
- Student access to view/download teacher resources
- Personal note storage for students
- Optional note sharing (with class or specific users)
- Assignment submission system (teacher-controlled permissions)
- Attendance tracking integration (optional, if not covered by existing ERP)

**Design Vision**
- Clean, intuitive interface for both teachers and students
- Organized hierarchy for classes, subjects, and materials
- Flexible permission system for content sharing

**Status**: Proposed

**Notes**  
*Attendance feature is optional depending on existing ERP integration requirements.*
